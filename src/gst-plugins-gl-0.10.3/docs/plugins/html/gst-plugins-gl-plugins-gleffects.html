Property Details

The "effect" property

  "effect"                   GstGLEffectsEffect    : Read / Write

Select which effect apply to GL video texture.

Default value: Do nothing Effect

The "hswap" property

  "hswap"                    gboolean              : Read / Write

Switch video texture left to right, useful with webcams.

Default value: FALSE
